基本的な動作

どう動くか（cssのプロパティと同じ表記）、何秒かけて動くか（duration）の2項目指定が必須。


gsap.to("オブジェクトを指定", { 
    動きのプロパティ: 数値 ,
    何秒かけるか: 秒数,
});

移動

【xまたはy ： 数値,】
各軸にどれくらい移動させるか

.js

copy 


gsap.to(".rect", { x: 200 ,duration: 1.3,});

色変更

【color ： 数値,】変更後の色

.js

copy 


gsap.to(".rect", {
    backgroundColor: "#abcfe0" ,
    duration: 1.3,
});

透明度変更

【opacity ： 数値,】1~0までの数値

.js

copy 


gsap.to(".rect", {
    opacity: 0.5 ,
    duration: 1.3,
});

gsap.to(".rect", {
