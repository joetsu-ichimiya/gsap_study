基本的な動作

どう動くか（cssのプロパティと同じ表記）、何秒かけて動くか（duration）の2項目指定が必須。


gsap.to("オブジェクトを指定", { 
    動きのプロパティ: 数値 ,
    何秒かけるか: 秒数,
});

移動

【xまたはy ： 数値,】

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
});

色変更

【color ： 数値,】変更後の色

.js

copy 


gsap.to(".rect", {
    backgroundColor: "#abcfe0" ,
    duration: 1.3,
});

透明度変更

【opacity ： 数値,】1~0までの数値

.js

copy 


gsap.to(".rect", {
    opacity: 0.5 ,
    duration: 1.3,
});

回転

【rotate ： 数値,】1~0までの数値

.js

copy 


gsap.to(".rect", {
    rotate: 50,
    duration: 1.3,
});

線

【border ： 色,】カラーコード

.js

copy 


gsap.to(".rect", {
    border: "#0000FF",
    duration: 1.3,
});

設定値

第2引数に設定値を含めることで、挙動をカスタマイズする。
リピート、イージングについては別記 


gsap.to("オブジェクトを指定", { 
    動きのプロパティ: 数値 ,
    何秒かけるか: 秒数,//ここが第2引数
    //追加で複数同時設定可
});

時間

【duration ： 秒数（デフォルト 1.0）,】

.js

copy 


gsap.to(".rect", { 
    x: 200 ,
    duration: 2.5,
});

遅延秒数

【delay ： 秒数（デフォルト 0）,】アニメーション開始前の待機時間

.js

copy 


gsap.to(".rect", {
    backgroundColor: "#abcfe0" ,
    duration: 1.3,
    delay: 1.5,
});

一時停止

【paused ： falseまたはtrue（デフォルト false）,】真偽値※tureでアニメーションを一時停止

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
    paused: true, // アニメーションを一時停止
});

上書き

【overwrite ： 真偽値,】falseで無効、trueで有効。autoも使用可能

.js

copy 


gsap.to(".rect", {
    rotate: 50,
    duration: 1.3,
    overwrite: true,
});

リピート

再生したの繰り返し設定


gsap.to("オブジェクトを指定", { 
    動きのプロパティ: 数値 ,
    何秒かけるか: 秒数,//ここが第2引数
    //追加で複数同時設定可
});

リピート回数

【duration ： 回数（デフォルト 0）,】
自然数を指定すると、その回数だけ繰り返す。
無限に繰り返したい場合は、repeat: -1を指定する

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
    repeat: 3,
});

リピートするまでの遅延


【repeatDelay ： 秒数（デフォルト 0）,】
リピートするまでの待機時間

.js

copy 


gsap.to(".rect", {
        x: 200,
        duration: 1.3,     
        repeatDelay: 2,
});

リピート時の反復挙動

【yoyo ： falseまたはtrue（デフォルト false）,】
真偽値※trueでアニメーションを一時停止

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
    paused: true, // アニメーションを一時停止
});

無限リピート

【repeat: -1,】
真偽値※trueでアニメーションを一時停止

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
    repeat: -1, // 無限リピート
});

リピート遅延

【repeatDelay: 秒数,】
真偽値※trueでアニメーションを一時停止

.js

copy 


gsap.to(".rect", {
    x: 200 ,
    duration: 1.3,
    paused: true, //アニメーション一時停止
});

タイムライン

タイムラインを設定して連続的なアニメーションを作成
関数を定義 


 //初期化
const 関数名 = gsap.timeline({リピートの設定});
//連続させる順番に追加していく
関数名.add(gsap.to("アニメーションさせる要素"),{動きの内容});
関数名.add(gsap.to("アニメーションさせる要素"),{次の動き});
            ・
            ・
            ・

.add(gsap.to())は省略可能 

const 関数名 = gsap.timeline({リピートの設定});
関数名.to("アニメーションさせる要素"),{動きの内容});
            ・
            ・
            ・

もしくは、

gsap.timeline({リピートの設定});
    .to("アニメーションさせる要素"),{動きの内容});
                                ・
                                ・
                                ・

モーションを重ねて、より滑らかな演出を行う

タイムラインのメソッド
タイムラインの作成で主に利用にするのは以下のメソッド。

関数名 説明
add() トゥイーン、タイムライン、コールバック、ラベルをタイムラインに追加します。
to() gsap.to()トゥイーンをタイムラインの最後に追加します。
from() gsap.from() トゥイーンをタイムラインの最後に追加します。
fromTo() gsap.fromTo() トゥイーンをタイムラインの最後に追加します。
set() タイミングで値をセットします。
call() タイミングで関数を呼び出します。
